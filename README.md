## View in Styleguidist

To view the available components in a browser:

```sh
npm run start
```

Then open [http://localhost:6060/](http://localhost:6060/).
This is based on [React Styleuidist](https://react-styleguidist.js.org/).

## Test

To test using jest:

```sh
npm run test
```
